== Description ==

Alex Is User Logged In allows you to easily display certain content to logged in users and not logged in users.
See http://anthony.strangebutfunny.net/my-plugins/alex-is-user-logged-in/ for help
 
== Installation ==

1. Upload `the directory "is-user-logged-in"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Is User Logged In" in the dashboard under Settings for help.
